== Description ==
DJ-Accessibility is a set of tools to help people with disabilities navigate the site. 
Use its' options to improve the accessibility of your website! You can display accessibility options in a pop-up or toolbar layout (available in the ver Pro) on your page.

DJ-Accessibility works with WordPress and it's also integrated with the YOOTheme Pro Builder.

Main features (Version Lite):
* Smart contrast (Invert colors, Monochrome, Dark contrast, Light contrast, Low saturation, High saturation, Highlight links, Highlight headings)
* Keyboard navigation 
* Mobile friendly
* Customizable icon
* Change the typo (font size, line height, and letter spacing)

Additional features available in the version Pro
* Toolbar layout
* Screen reader
* Read mode

== Installation ==

1. In your admin panel, go to Plugins -> Add New and click the Upload Plugin button.
2. Click Choose File, then select the plugin ZIP file. Click Install Now.
3. Click Activate to activate the plugin.
